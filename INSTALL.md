# YouDown 🎬

> Скачивай видео с YouTube в один клик. Расширение для браузера + yt-dlp.

## Как это работает

1. На странице YouTube нажимаешь на иконку расширения **YouDown**
2. Открывается окошко (popup) с выбором качества и кнопкой **"Скачать"**
3. Браузер отправляет ссылку в **yt-dlp** (через Windows Protocol `ytdlp://`)
4. yt-dlp сам обновляется (если нужно) и скачивает видео
5. Готовое видео — в папке `~/Downloads/YouDown/`

## Установка

### Шаг 1. Установить yt-dlp и ffmpeg

Открой **Командную строку** (Win+R → `cmd` → Enter) и выполни:

```cmd
winget install yt-dlp.yt-dlp
winget install ffmpeg
```

> Если `winget` не работает — скачай [yt-dlp.exe](https://github.com/yt-dlp/yt-dlp/releases/latest) и [ffmpeg.exe](https://ffmpeg.org/download.html) и положи в `C:\Windows\System32` (или добавь в `PATH`).

Проверь, что установилось:

```cmd
yt-dlp --version
ffmpeg -version
```

> **ffmpeg** нужен для склейки видео и аудио дорожек в один файл. Без него видео и аудио скачаются отдельными файлами.

---

### Шаг 2. Скопировать файлы куда-нибудь

Скопируй папку проекта (в репозитории она называется `youdown`) в удобное место, например:

```
C:\Program Files\YouDown\
```

В этой папке должны быть:

```
C:\Program Files\YouDown\
├── ytdlp-handler.bat     ← входная точка (вызывается браузером)
├── handler.ps1           ← PowerShell-обработчик (парсит URL, запускает yt-dlp)
└── register-protocol.reg ← регистрация протокола в Windows
```

---

### Шаг 3. Зарегистрировать протокол ytdlp://

Дважды кликни `register-protocol.reg` и подтверди "Да".

**Важно:** Если ты положил .bat не в `C:\Program Files\YouDown\`, открой .reg в Блокноте и исправь путь:

```diff
- "C:\\Program Files\\YouDown\\ytdlp-handler.bat" "%1"
+ "D:\\МоиПрограммы\\YouDown\\ytdlp-handler.bat" "%1"
```

> Обрати внимание: обратные слеши экранируются — пиши `\\`, а не `\`.

---

### Шаг 4. Установить расширение

1. Открой **Yandex Browser** (или Chrome)
2. Перейди на страницу расширений: `browser://extensions/`
3. Включи **"Режим разработчика"** (справа сверху)
4. Нажми **"Загрузить распакованное расширение"**
5. Выбери папку `extension/` из проекта

Готово! Расширение появится в браузере.

### Шаг 5. Проверка

1. Открой любое видео на YouTube
2. Нажми на иконку расширения **YouDown** (справа в адресной строке)
3. В открывшемся окошке выбери качество и нажми **"Скачать"**
4. Откроется консоль, yt-dlp скачает видео
5. Готовый файл — в `~/Downloads/YouDown/`

Если консоль не появляется — проверь, что `yt-dlp` установлен и .reg-файл зарегистрирован.

## Выбор качества

| Опция | Формат (аргумент `-f` для yt-dlp) | Описание |
|-------|--------------------------------------|---------|
| Наилучшее видео | `bestvideo+bestaudio/best` | Самое лучшее доступное |
| 1080p Full HD | `bestvideo[height<=1080]+bestaudio/best[height<=1080]` | Full HD видео |
| 720p HD | `bestvideo[height<=720]+bestaudio/best[height<=720]` | HD видео |
| 480p | `bestvideo[height<=480]+bestaudio/best[height<=480]` | Среднее качество |
| 360p | `bestvideo[height<=360]+bestaudio/best[height<=360]` | Низкое качество |
| Только аудио (MP3) | `bestaudio/best` | Только звук (конвертируется в MP3) |

## Требования

- **Windows** (10/11; на Windows 7 winget не работает — скачай .exe вручную)
- **Yandex Browser** или **Google Chrome** (любой Chromium-браузер)
- **yt-dlp** и **ffmpeg** (устанавливаются одной командой `winget install`)
- **Права администратора** — только для регистрации протокола (один раз)

## Как обновлять?

Расширение — просто окошко с выбором качества, обновлять не нужно. При обновлении файлов `.ps1` / `.bat` просто скопируй свежие файлы из папки `windows/`.

yt-dlp обновляется автоматически **перед каждым скачиванием**.

## Возможные проблемы

| Проблема | Решение |
|----------|---------|
| После нажатия "Скачать" ничего не происходит | Проверь, зарегистрирован ли протокол: дважды кликни `register-protocol.reg` |
| Консоль открывается и сразу закрывается | Обнови `handler.ps1` — старая версия могла не найти yt-dlp |
| "yt-dlp not found" при установленном yt-dlp | Обнови `handler.ps1` — добавлен автоматический поиск в папке winget |
| Видео скачивается, но файл без названия (в имени `%(title)s`) | Обнови `handler.ps1` — исправлен шаблон имени файла |
| Видео и аудио отдельными файлами | Установи `ffmpeg`: `winget install ffmpeg` |
| "ffmpeg not found" | Установи `winget install ffmpeg` или скачай ffmpeg.exe вручную |

## Удаление

1. На `browser://extensions/` нажми "Удалить" на YouDown
2. Удали папку с файлами
3. (Опционально) удали запись из реестра: открой `regedit`, найди `HKEY_CLASSES_ROOT\ytdlp` и удали
